Designed a Graduate Program web Application using Maven archetype, ORM Mapping, HQL queries. Also handled Subversion repository and implemented Security features. It deals with creating wallet that is responsible for making bitcoin transaction, register user and making them authorized for making transactions and display. Dealing with address and labeling of addresses assigned to authorized user for making transactions.

Environment: Spring, Hibernate and PostgreSQL, SVN,Git .
