AutoApply Job BOT

source code
Link

Designed a automation bot using webdriver to getting all job posting from popular job websites like Linkedin, Monster , Indeed and adding an Extra feature of Auto Apply using NightwatchJS. I am using cheerio and request module for sending request to all the scraped URLs and performing necessary events such as login and uploading resume for applying to the job opening, Moreover have tried to make it user friendly by making it GUI rich. It takes your account details and apply on each job posting one by one.

Environment: nightwatch , NodeJS, Selenium-Web-server and other npm modules (NightmareJS, SlimerJs, Loadash and etc.) , Express , HTML5/CSS ,angular , javascript, jquery, NoSQl, JSON, Api, Git, Heroku.
