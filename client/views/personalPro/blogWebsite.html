Developed my own Profile website, listing all my skills and projects with feature of blogging and review , furthermore adding functionality of crypto-currency payments. This a blogging webite with feature of posting a blog , with autheication of User by google and email verification. It allows user to login and singup and post a blog, moreover has job postings from liked in. Extensive use of firebase and angular . For UI i am using angular material. Auth using Google Firebase .Moreover have included sample for using angular fire, created Restful Api for website data. Deployment with Heroku and Github.

Environment: Firebase, NODE js , Express , HTML5/CSS , SASS, ng-material , angular , javascript, jquery, NoSQl, JSON, Api, Git, Heroku.
